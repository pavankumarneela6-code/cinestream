# 🎬 Movie Streaming Website

## 📖 Project Description

Movie Streaming Website is a front-end web application built using HTML and CSS. The website provides a modern streaming platform interface where users can browse trending movies, popular movies, TV shows, upcoming releases, subscription plans, recommendations, and continue watching content.

## 🚀 Features

* Responsive Navigation Bar
* Search Movie Functionality (UI)
* User Profile Section
* Featured New Movies Banner
* Trending Movies Section
* Popular Movies Grid Layout
* Top Rated Movies
* Latest Releases
* Movie Categories
* Continue Watching Section
* Recommended Movies Section
* TV Shows with Seasons & Episodes
* Upcoming Movies Section
* Subscription Plans
* User Reviews
* Footer with Social Media Links
* Hover Effects and Animations
* Horizontal Scrolling Movie Lists

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Hover Effects
* Responsive Design Concepts
